## ?? Como Conquistar Cada Badge

### ? Entrega no Prazo
```
? TODO trabalho deve estar entregue
? Nem um atrasado
? Resultado: Automaticamente desbloqueado
```

### ?? Notas Altas
```
? Média geral = (nota1 + nota2 + ...) / total
? Deve ser ? 9.5
? Exemplo: 9.5, 9.8, 10.0 = Conquistado
? Exemplo: 9.4, 9.3, 8.9 = Bloqueado
```

### ? Envio Consistente
```
? TODOS os trabalhos devem ter algo enviado
? Nem um trabalho vazio
? Resultado: Automaticamente desbloqueado
```

### ?? Engajamento Diário
```
? Precisa estar ativo todos os dias
? Enviar algo todos os dias
? Resultado: Automático (futuro)
```

### ?? Top Performer
```
? Ter PELO MENOS 3 notas ? 9
? Exemplo: 9.0, 9.5, 10.0 = Conquistado
? Exemplo: 8.9, 9.1, 9.2 = Bloqueado
```

### ?? Participaçăo Ativa
